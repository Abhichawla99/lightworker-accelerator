# Image Assets for Lightworker Accelerator Landing Page

Add Sarah's images to this folder with these recommended filenames:

## Required Images:

1. **sarah-hero.jpg** (or .png)
   - Hero section image of Sarah
   - Professional, warm, approachable
   - High quality (at least 1200px wide)
   - Should convey authority and spirituality

2. **sarah-about.jpg** (or .png)
   - Image for "About Sarah" section
   - Can be more casual/personal
   - 800-1000px wide recommended

3. **sarah-teaching.jpg** (or .png)
   - Sarah in action (teaching, speaking, or guiding)
   - Shows her in her element
   - 800-1000px wide

4. **sarah-testimonial-bg.jpg** (or .png)
   - Background image for testimonial section
   - Can be softer/more ethereal
   - 1200px+ wide

## Optional Images:

5. **sarah-portrait.jpg** - Additional portrait option
6. **sarah-lifestyle-1.jpg** - Lifestyle/brand images
7. **sarah-lifestyle-2.jpg** - More lifestyle content
8. **logo.png** - If Sarah has a logo

## Image Guidelines:
- Use JPG for photos, PNG for images with transparency
- High resolution (at least 1200px wide for full-width images)
- Professional quality
- Natural lighting preferred
- Warm, inviting tones that match the site's color palette
- Images should convey: confidence, authenticity, spirituality, groundedness

Once you add the images, the page will automatically display them!
